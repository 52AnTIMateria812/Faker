# Техническое задание на проект "Демонстрационный стенд библиотеки Faker"

## 1. Общая информация

**Название проекта:** Демонстрационный стенд библиотеки Faker  
**Цель проекта:** Создать консольное приложение для наглядной демонстрации возможностей библиотеки `Faker` (генерация случайных реалистичных данных) в Python.

## 2. Описание задачи (что нужно сделать)

Необходимо разработать модульное консольное приложение, которое в интерактивном режиме демонстрирует практическое применение ключевых генераторов модуля `Faker`. Приложение должно показывать, как легко создавать профили, адреса, данные компаний и платежную информацию.

## 3. Функциональные требования

1. Система должна предоставлять текстовое меню для выбора генератора.
2. Система должна демонстрировать генерацию реалистичных профилей.
3. Система должна демонстрировать генерацию случайных адресов.
4. Система должна демонстрировать генерацию атрибутов компании (название, должность, слоган).
5. Система должна демонстрировать генерацию данных кредитных карт.
6. Система должна корректно обрабатывать неверный ввод пользователя.
7. Система должна использовать русскую локаль `Faker('ru_RU')` для большей наглядности.

## 4. Архитектура и структура проекта

```
Faker/
├── core.py          # Основные функции с инициализацией Faker
├── utils.py         # Вспомогательные функции и декораторы (@timer, print_separator)
├── logic.py         # Логика демонстраций
├── app.py           # Главное приложение с меню
├── requirements.txt # Зависимости
└── README.md        # Документация проекта
```

### 1. `core.py` - Ядро генерации
Содержит объект `fake` и функции `generate_profile`, `generate_address` и т.д.

### 2. `utils.py` - Утилиты
- **`timer(func)`** - декоратор для замера скорости работы `Faker`.
- **`print_separator(title)`** - функция для красивого вывода в консоль.

### 3. `logic.py` - Логика
Обертки над функциями ядра, которые выводят результаты в удобном `print`.

### 4. `app.py` - Главное меню
Отвечает за консольный инпут и навигацию (`show_menu()`, `main()`).

## 5. Запуск
```bash
pip install -r requirements.txt
python app.py
```
